# 🚀 Bixbott

> Next-generation AI Agent IDE for building, reviewing, and deploying applications.

![Bixbott Banner](https://placehold.co/1200x400/0f172a/ffffff?text=Bixbott)

## ✨ Giới thiệu

Bixbott là một nền tảng Agent IDE thế hệ mới được thiết kế để hỗ trợ lập trình viên xây dựng ứng dụng nhanh hơn bằng sức mạnh của AI.

Nó kết hợp:

- 🤖 AI Coding Agent
- 🧠 Code Generation
- 🔍 Code Review
- 🚀 Deployment Assistant
- 📦 Project Scaffolding
- 🔄 GitHub Integration

Mục tiêu của Bixbott là trở thành một môi trường phát triển hiện đại tương tự Cursor, Windsurf và OpenCode nhưng tập trung vào khả năng Agent tự động xây dựng ứng dụng.

## ⚡ Tính năng

### Agent Build

npm run dev
